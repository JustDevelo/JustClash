# 简介

JustClash 是一款简洁、快速、无广告的 Windows 代理客户端。基于 **Mihomo (Clash.Meta)** 内核，内核已内置于应用中。

# 功能

- **全局 TUN 模式** — 保护整个系统的流量
- **一键导入订阅**（支持 VLESS 等协议、Clash 与原始配置）
- **自动更新订阅** — 启动时与后台自动更新
- **服务器 / 节点选择**、代理分组、延迟测试、国家旗帜
- **规则模式与直连模式**
- **内置 Geo 数据库**，自动更新
- **主题**：浅色 / 深色 / 纯黑（AMOLED）+ 强调色
- **语言**：俄语、英语、中文
- **系统托盘**、开机自启、单实例运行
- **低资源占用** — 低配置电脑也能流畅运行

# 安装

1. 从 [Releases](https://github.com/JustDevelo/JustClash/releases) 页面下载最新的 `JustClash-Setup-x.x.x.exe`。
2. 运行并按安装向导操作。

# 技术栈

- 界面：Flutter (Dart)
- 内核：[Mihomo (Clash.Meta)](https://github.com/MetaCubeX/mihomo)
